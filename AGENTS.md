# Project Instructions

- Do not add compatibility migrations for removed or malformed config values. Let invalid saved configs fail validation and reset to defaults through `CCAutomated.loadConfig`.
- `references/cookie-clicker-main.js` may exist as a local-only copy of Cookie Clicker's source for implementation reference. It contains third-party copyrighted code, is intentionally ignored by git, and should not be committed.
- Edit authored addon code in `src/`: `core/` for setup and lifecycle, `automation/` for features that act on Cookie Clicker, `strategy/` for reusable decision logic, and `ui/` for menu/status rendering. Root `main.js` is generated by `npm run build` and is committed only so the bookmarklet can load one file.
